# Fitkle No-Show Prevention MVP

AI 기반 밋업 노쇼 방지 솔루션

## 🎯 핵심 가설

**"AI 에이전트가 능동적으로 참석 확인을 하면 노쇼율이 줄어든다"**

## 🚀 핵심 기능

### 1. 능동형 생존 확인 (Active Check)
- D-1일 전 참석 확인 버튼 발송
- 미응답 시 자동 취소 + 대기자 호출

### 2. 대기자 자동 매칭
- 취소 발생 시 대기자에게 즉시 알림
- 10분 내 응답으로 자동 확정

### 3. 라스트 미닛 협상
- 임박 취소 시 대기자 호출 가능 여부 확인
- 결원 최소화

## 📊 검증 지표

- **Before**: 기존 방식 노쇼율 측정
- **After**: AI Agent 개입 후 노쇼율 측정
- **목표**: 노쇼율 80% 이상 감소

## 🏗️ 기술 스택

- Frontend: Next.js (React)
- Backend: Next.js API Routes
- Database: Supabase (PostgreSQL)
- AI Agent: OpenAI API
- 알림: 카카오 알림톡 / 이메일
